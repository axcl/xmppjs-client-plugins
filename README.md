* [XEP-0085 - Chat State Notification](https://xmpp.org/extensions/xep-0085.html)
* [XEP-0096 - SI File Transfer](https://xmpp.org/extensions/xep-0096.html)
* [XEP-0384 - OMEMO](https://xmpp.org/extensions/xep-0384.html)
